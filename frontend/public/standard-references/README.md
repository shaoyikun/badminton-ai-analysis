# standard-references

这个目录用于存放前端报告页展示的“标准动作参考素材”。

当前策略：
- 允许混合使用两类素材：
  1. `*.svg`：结构示意图 / 占位参考图
  2. `*.jpg` / `*.png`：真人标准动作关键帧
- 当某个动作还没有足够可靠的真人标准素材时，可以先用示意图占位，但要在后续迭代中逐步替换。

当前文件：
- `clear-reference.svg`：正手高远球示意参考图（保留作兜底占位）
- `clear-reference-real.jpg`：正手高远球第一版真人参考帧（主参考）
- `clear-phase-prep.jpg`：高远球阶段参考帧 - 准备
- `clear-phase-contact.jpg`：高远球阶段参考帧 - 击球
- `clear-phase-follow.jpg`：高远球阶段参考帧 - 收拍
- `smash-reference.svg`：杀球示意参考图（保留作兜底占位）
- `smash-reference-real.jpg`：第一版真人杀球参考帧（单帧）
- `smash-phase-prep.jpg`：杀球阶段参考帧 - 准备
- `smash-phase-load.jpg`：杀球阶段参考帧 - 引拍 / 起跳加载
- `smash-phase-contact.jpg`：杀球阶段参考帧 - 击球

当前状态：
- 高远球、杀球两个 MVP 动作都已具备“主参考帧 + 阶段性参考帧”的最小素材组
- SVG 示意图暂时继续保留，作为真人素材不足时的兜底方案

后续建议：
- 为每个动作继续补更多角度和更干净背景的真人参考帧
- 逐步把当前小型素材组升级成可扩展的标准动作素材库
