# samples

这个目录用于放本地联调 / 演示用的视频样本。

## 当前策略
- 默认**不纳入 Git 版本控制**（避免仓库变大、版权不清、二进制频繁变动）
- 本地开发时可自行放入测试视频
- 如果未来需要对外分发 demo 素材，建议单独准备一套明确可公开使用的样本，并在文档中说明来源与用途

## 当前已用到的样本类型
- `clips/`：适合直接跑 PoC 的短视频片段
- `raw/`：原始长视频素材（如下载的公开视频）

## 协作约定

- 默认使用本地自备视频，不要求把样本提交到仓库
- 用于 smoke test / 回归验证的样本，建议满足以下条件：
  - 5 到 15 秒
  - 单人出镜
  - 固定机位
  - 能完整覆盖一次可分析动作
- 如果团队需要共享演示素材，建议在仓库外部维护一套明确可公开使用的样本，并在这里补充获取方式

## 回归建议

- `clips/`：适合快速验证上传、分析和结果页链路
- `raw/`：适合手动裁剪新的回归样本
- 交付手册中的最小回归路径默认引用这里的本地样本策略
