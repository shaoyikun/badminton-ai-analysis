# PRODUCT-SPEC

## 1. 产品定位
羽毛球 AI 动作分析与训练建议平台，首发面向移动端。用户通过手机上传羽毛球动作视频，系统输出动作问题诊断、训练建议和复测路径。

## 2. 产品目标
- 让用户看懂自己的动作问题
- 让用户知道问题会如何影响击球表现
- 让用户得到可执行的训练建议
- 让用户形成训练—复测—进步跟踪闭环

## 3. 平台约束
- 首发仅考虑移动端上线
- 默认手机竖屏场景
- 交互、信息密度、上传链路均以移动端优先
- 前端实现限定在 React 框架体系内
- 前端长期路线按 React Web / React H5 规划，当前阶段不采用 React Native 作为默认实现路线

## 4. MVP 范围
### 支持动作
1. 正手高远球
2. 杀球

### 输入约束
- 单人视频
- 5~15 秒
- 固定机位
- 推荐侧后方 / 正后方视角
- 一段视频仅分析一种动作

### 输出结果
- 动作类型
- 总评分
- 分维度评分
- Top 3 问题
- 问题影响说明
- 对应训练建议
- 标准动作关键帧对比
- 复测建议

## 5. 用户主流程
首页 → 首次引导 → 拍摄规范 → 上传 → 上传确认 → 分析中 → 分析完成 → 报告页 → 复测对比

## 6. 异常流程
- 上传失败
- 识别失败
- 多人同框
- 机位不合适
- 时长不符合要求

## 7. 商业化假设
### C 端
- 免费 1 次基础分析
- 付费解锁更详细报告 / 多次上传 / 历史分析 / 训练计划

### 后续 B 端
- 教练课后报告
- 球馆 / 俱乐部学员成长档案
